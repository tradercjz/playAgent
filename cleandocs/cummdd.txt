

### 语法

`cummdd(X, [ratio=true])`

### 详情

累计计算传入向量数据的最大回撤。

### 参数

- **X** 数值向量，表示用于计算最大回撤的输入数据，一般指累积的收益或收益率。注意：不可为空。
- **ratio** 布尔标量，表示是否返回最大回撤率。默认值为 true。
  * true：返回最大回撤率，计算公式为：  
    ![](../images/mdd1.png)
  * false：返回基于下降金额绝对值的最大回撤，计算公式为：  
    ![](../images/mdd2.png)

### 示例

假设投资组合累积收益数据如下：
```DolphinDB
x = [36, 96, 42, 100, 59, 86, 25, 64, 72]

// 计算最大回撤率
cummdd(x)
// 输出：[0,0,0.5625,0.5625,0.5625,0.5625,0.75,0.75,0.75]

// 计算绝对回撤值
cummdd(x, false)
// 输出：[0,0,54,54,54,54,75,75,75]
```
