

### trueRange 函数文档

#### 语法
```DolphinDB
trueRange(high, low, close)
```

#### 参数
- **high**: 数值型向量，表示最高价序列
- **low**: 数值型向量，表示最低价序列
- **close**: 数值型向量，表示收盘价序列

#### 功能说明
计算真实波幅（True Range），结果为以下三个值的最大值：
1. 当日最高价与最低价之差绝对值 `|high - low|`
2. 当日最高价与前一日收盘价之差绝对值 `|high - prev_close|`
3. 前一日收盘价与当日最低价之差绝对值 `|prev_close - low|`

#### 示例
```DolphinDB
// 创建包含股票数据的内存表
sym = take(`A`B, 10)
date = 2020.01.01 2020.01.01 2020.01.02 2020.01.02 2020.01.03 2020.01.03 2020.01.04 2020.01.04 2020.01.05 2020.01.05
high = [11.48, 10.23, 12.26, 10.7, 12.24, 10.45, 12.3, 10.51, 12.24, 10.49]
low = [10.91, 9.41, 11.18, 10, 11.71, 9.83, 11.62, 9.91, 11.1, 9.6]
close = [11.38, 10.22, 12.1, 10.31, 11.89, 10.21, 12.13, 10.47, 11.35, 9.81]
t = table(sym, date, high, low, close)

// 按股票代码分组计算真实波幅
select sym, trueRange(high, low, close) as trange 
from t 
context by sym
```

#### 输出结果
| sym | trange |
|-----|--------|
| A   |        |
| A   | 1.08   |
| A   | 0.53   |
| A   | 0.68   |
| A   | 1.14   |
| B   |        |
| B   | 0.7    |
| B   | 0.62   |
| B   | 0.6    |
| B   | 0.89   |

#### 实现说明
- 输入向量长度需保持一致
- 计算结果首行为空（因需前一日收盘价）
- 支持通过 `context by` 实现分组计算
