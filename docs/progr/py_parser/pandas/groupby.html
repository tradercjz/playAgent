GroupBy

GroupBy 类是用于数据分组和聚合操作的关键工具。它允许用户按照一个或多个列的值将数据集拆分成多个组，可以对每个分组进行聚合、转换和分析操作。

使用 DolphinDB pandas GroupBy 时需要注意以下问题：

  • groupby 的列数据必须类型一致。
  • 除 cumcount 外的其它函数分组计算后的结果和原始输入数据的顺序不同。结果中组内的数据和原始数据顺序保持一致，但每组之间按照 groupby 的列排序。
