S01022

错误代码

S01022

报错信息

Cannot back up the partitions to the specified backupDir as back up files created with SQL statement already exist in the directory. RefId:S01022

错误原因

通过分区方式进行备份时，备份目录下已存在通过 SQL 方式备份的文件。

解决办法

检查备份目录是否存在 SQL 方式备份的文件，或换用新目录。
