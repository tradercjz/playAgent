S10004

报错信息

GPLearn currently does not support function <xxx>.RefId: S10004

错误原因

在自定义 fitness 或使用自定义函数训练中，使用了 GPLearn 不支持的函数。

解决办法

参阅 GPLearn 用户手册，使用 GPLearn 当前支持的函数进行训练或计算 fitness。
